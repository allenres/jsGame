# Botanical Clicker

A modern incremental game where you nurture a plant from a sprout to a towering tree.

## 🌿 Game Mechanics

* **Manual & Passive Growth**: Increase the plant's height by clicking the **Nourish Plant** button or by waiting for the **1 inch/second** auto-increment.
* **Visual Evolution**: The plant's image dynamically updates as you hit height milestones (100, 500, and 1,000 inches), moving through four growth stages.
* **Garden Shop**: Spend accumulated height to buy tools like **Watering Cans** and **Hoses**.
* **Exponential Scaling**: Each purchase increases your **Inches per click** bonus but also causes the item's cost to **double**.
